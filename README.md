# Traffic Violation Detection System

End-to-end system for detecting vehicle speed violations using YOLOv8, DeepSORT, EasyOCR, Django REST API, Celery, and React.

## Phases
1. Environment Setup
2. Vehicle Detection (YOLOv8)
3. Vehicle Tracking (DeepSORT)
4. Speed Detection
5. License Plate Detection
6. OCR (EasyOCR)
7. Violation Pipeline
8. Database (PostgreSQL)
9. Django REST API
10. Async Processing (Celery + Redis)
11. Dashboard (React)
12. Deployment (Docker + AWS)
